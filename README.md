# Maria Talks Tech

Static site with Tailwind CSS. Local development uses Tailwind CLI and `live-server`.

## Quick start

```bash
cd maria-talks-tech   # or your project directory
npm install
npm run dev
```

Then open `http://127.0.0.1:8080/`. The server watches `public/` and `src/`.

## Scripts

- `npm run dev`: runs Tailwind in watch mode and starts `live-server` on port 8080.
- `npm run build`: compiles `src/styles/input.css` → `public/assets/styles.css` (minified).
- `npm run format`: formats files with Prettier.

## Project layout

```
public/                # Static assets served in dev and deployed as-is
  assets/styles.css    # Generated by Tailwind CLI (gitignored)
  index.html           # Main page
  work-with-me.html    # Work With Me page
  images/, icons/      # Static assets
src/
  styles/input.css     # Tailwind source (@import "tailwindcss")
  scripts/main.ts      # Placeholder for future interactivity
```

**Tailwind:** HTML files use compiled `assets/styles.css`. Run `npm run build` before deploying.

## Requirements

- Node.js 18+ and npm
- macOS/zsh environment (project rules assume this)

## Publishing to GitHub

The repo is configured to use SSH by default:

```
git remote -v
# origin  git@github.com:mashablair/maria-talks-tech.git
git push -u origin main
```

If you haven’t added an SSH key yet:
1) `ssh-keygen -t ed25519 -C "you@example.com"`
2) Add the public key (`~/.ssh/id_ed25519.pub`) at GitHub → Settings → SSH and GPG keys.
3) Test: `ssh -T git@github.com`

## Deploy options

### Cloudflare Pages (Git integration)

1. Go to [dash.cloudflare.com](https://dash.cloudflare.com) → Workers & Pages → Create → Pages → Connect to Git.
2. Select `mashablair/maria-talks-tech` and authorize.
3. Configure:
   - **Framework preset:** None
   - **Build command:** `npm run build`
   - **Build output directory:** `public`
   - **Root directory:** `/` (default)
4. Deploy. Each push to `main` triggers a new deploy.

### Other platforms

- **GitHub Pages / Netlify / Vercel:** Output directory `public/`, build command `npm run build`.
